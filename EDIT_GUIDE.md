## ⚙️ How to Edit

1. **Open the file:** `script.js` in VS Code
2. **Find the line** you want to edit (numbers shown above)
3. **Keep the quotes:** Everything goes inside single quotes `'like this'`
4. **Use line breaks:** Use `\n` for new lines
   - Example: `'Line 1\nLine 2\nLine 3'`
5. **Save the file** (Ctrl+S)
6. **Refresh browser** to see changes
